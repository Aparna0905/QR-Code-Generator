# QR Code Generator

A simple and responsive QR Code Generator built using HTML, CSS, and JavaScript.
This project generates QR codes instantly from user input using a free QR code API.

## Features

* Generate QR codes instantly
* Simple and clean UI
* Input validation with error animation
* Lightweight and beginner-friendly
* Responsive design

## Technologies Used

* HTML
* CSS
* JavaScript
* QR Server API

## How It Works

1. Enter any text or URL in the input field.
2. Click the generate button.
3. The QR code will be displayed instantly.

## API Used

```text
https://api.qrserver.com/v1/create-qr-code/
```
